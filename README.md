# taller3LabFitServer
Taller 3 del laboratorio de sistemas operativos.
código para conectar uno o más clientes de un chat.
Ejecutarlo en entorno de red de en donde los usuarios
deben esperar mensajes del o los clientes y los reenvía a todos los clientes en la misma sala
usando un bucle “for” sobre la lista de clientes en esa sala.
Finalmente, cuando el cliente se desconecta, el hilo de comunicación se cierra y se elimina al usuario de la lista de clientes
en la sala correspondiente.
